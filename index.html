📖 Инструкция по работе с приложением (нажмите, чтобы развернуть)

Лазерная насадка работает в 2 режимах:

  • детектирование по пьезоэлементу;
  • ожидание команды (ручное моргание лазером);

При нажатии кнопки "Импульс" насадка испускает лазерный импульс и переходит в режим ожидания команды.

При нажатии кнопки "Пьезо" насадка переходит в режим работы по пьезоэлементу (команда 0x03).

При нажатии кнопки "Сброс" насадка переходит в режим работы по пьезоэлементу со следующими настройками:

  • Время импульса:100 мс;
  • Время паузы:1000 мс;

В блоке телеметрии отображены текущие настройки насадки.

Чтобы обновить настройки насадки(время импульса/паузы) необходимо заполнить поля в блоке "Настройка" и нажать кнопку "Обновить настройки".

При обновлении настроек насадка автоматически переходит в режим работы по пьезоэлементу.

При включении Микрофона (функция находится в разработке) насадка переходит в режим ожидания команды с испусканием лазерного импульса с текущими настройками.
